## ğŸš€ Quick Start Guide

### Step 1: Take Screenshots
1. Install the extension locally in Opera GX or Chrome
2. Open `store-assets/demo-page.html` in browser
3. Enable HighlightAssist
4. Take 3-5 screenshots showing:
   - Extension popup UI
   - Element highlighting in action
   - Badge indicator
5. Save to `store-assets/screenshots/` folder

### Step 2: Generate Promotional Graphics
1. Open `store-assets/promotional-graphics.html` in browser
2. Use browser screenshot tools to capture:
   - Large tile (1280x800) for Chrome
   - Small tile (640x400) for Chrome
3. Save as `promo-large.png` and `promo-small.png`

### Step 3: Submit to Chrome Web Store
1. Go to [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole)
2. Pay $5 registration fee (one-time)
3. Upload `dist/HighlightAssist-v1.0.0.zip`
4. Copy content from `STORE_LISTING.md` (Chrome section)
5. Upload screenshots and promotional graphics
6. Submit for review

### Step 4: Submit to Opera Addons
1. Go to [Opera Addons Developer](https://addons.opera.com/developer/)
2. No fee required!
3. Upload `dist/HighlightAssist-v1.0.0.zip`
4. Copy content from `STORE_LISTING.md` (Opera section)
5. Upload screenshots
6. Submit for review

### Step 5: Submit to Firefox Add-ons
1. Go to [Firefox Developer Hub](https://addons.mozilla.org/developers/)
2. No fee required!
3. Upload `dist/HighlightAssist-v1.0.0.zip`
4. Copy content from `STORE_LISTING.md` (Firefox section)
5. Upload screenshots
6. Submit for review
