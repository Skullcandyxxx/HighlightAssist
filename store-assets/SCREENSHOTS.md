# Screenshot Instructions

## Required Screenshots for Store Submissions

### Chrome Web Store
- **Required:** 1-5 screenshots
- **Dimensions:** 1280x800 or 640x400
- **Format:** PNG or JPEG

### Opera Addons
- **Required:** At least 1 screenshot
- **Recommended:** 3-5 screenshots
- **Max dimensions:** 1280x800
- **Format:** PNG or JPEG

### Firefox Add-ons (AMO)
- **Required:** At least 1 screenshot
- **Recommended:** 3-5 screenshots
- **Dimensions:** Any, but 1280x800 recommended
- **Format:** PNG or JPEG

---

## How to Create Screenshots

### Option 1: Manual Screenshots (Recommended)

1. **Install the extension** in Opera GX or Chrome
2. **Navigate to a Vite project** (e.g., http://localhost:5173)
3. **Enable the extension** by clicking the icon
4. **Take screenshots** showing:
   - Extension popup (on/off state)
   - Element being highlighted on hover
   - Element details being displayed
   - Badge showing "ON" state

## Screenshot Naming Convention

Save screenshots as:
- `screenshot-1-popup.png` - Extension popup interface
- `screenshot-2-highlighting.png` - Element being highlighted
- `screenshot-3-fullpage.png` - Full browser context
- `screenshot-4-badge.png` - Badge indicator
- `screenshot-5-keyboard.png` - Keyboard shortcut demo

## Quick Screenshot Tool (Windows)

Use Windows Snipping Tool or Snip & Sketch:
- Press `Win + Shift + S`
- Select area to capture
- Image is copied to clipboard
- Paste into Paint or image editor
- Save as PNG

Or use browser DevTools:
- Press F12
- Ctrl+Shift+P (Command Palette)
- Type "screenshot"
- Choose "Capture full size screenshot"
