# Icon Placeholders

To complete the extension, you need PNG icons in these sizes:
- icon16.png (16x16)
- icon32.png (32x32)  
- icon48.png (48x48)
- icon128.png (128x128)

## Quick Solution

Use an online SVG to PNG converter with the `icon.svg` file:
1. Go to https://www.svgtopng.com/
2. Upload `icon.svg`
3. Generate PNGs at 16px, 32px, 48px, and 128px
4. Save them here with the correct names

## Or use a design tool

- Figma
- Canva
- Adobe Illustrator
- Inkscape (free)

The current icon design shows a magnifying glass with a pointer - representing the inspect/highlight functionality.
