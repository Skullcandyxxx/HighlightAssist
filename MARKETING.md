## Code Example

The injected script dynamically imports your Vite module:

```javascript
// Simplified version of injected.js
async function loadHighlightTool() {
  const ports = [5173, 3000, 3001, 5174];
  
  for (const port of ports) {
    try {
      const module = await import(`http://localhost:${port}/@vite/client`);
      // Module loaded successfully, Vite server found!
      await import(`http://localhost:${port}/src/highlight/highlight-tool.js`);
      console.log('‚úÖ HighlightAssist loaded');
      break;
    } catch (error) {
      continue; // Try next port
    }
  }
}
```
