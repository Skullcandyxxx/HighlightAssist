Thank you for trying HighlightAssist Bridge — the secure desktop companion that lets the browser extension stream UI insights to your IDE.

Before you continue, please review the key terms below:

• Requirements
  - Windows 10 or newer with an active internet connection.
  - Python 3.8+ installed globally and added to PATH (the installer will verify this).
  - You must have permission to install software on this device.

• Privacy & Telemetry
  - HighlightAssist Bridge runs locally and only relays UI metadata that you explicitly capture.
  - No analytics, telemetry, or third-party calls are performed without your action.
  - Log files stay on your machine in %LOCALAPPDATA%\HighlightAssist.

• Security & Network Usage
  - The bridge opens a localhost WebSocket (default port 5055). No public ports are exposed.
  - Always close the bridge when you are finished sharing UI data with AI assistants.
  - Review scripts before running them in regulated or air‑gapped environments.
